The below commands should be executed from a command window in working directory "stroke-app-ionic".

1) run 'npm run init' to install prject dependencies and add android platform to ionic project

2) run 'ionic state reset --plugins' to install all plugins

3) To run locally execute 'npm run runlocal'
	NOTE: Ionic serve runs live reload on file changes

4) To run on debug enabled android device run 'npm run runAndroid'
