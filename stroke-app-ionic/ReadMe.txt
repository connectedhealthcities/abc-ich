This is an ionic project.

Available ionic commands can be listed at the command prompt using the following command:

ionic help

Commonly used commands in development are:

ionic serve
ionic run android
gulp test

ionic serve
===========
Runs the code in a desktop browser.
Additionally, this command starts LiveReload which is used to monitor changes in the file system.
As soon as you save a file the browser is refreshed automatically.

ionic run android
=================
Runs the code on a connected device.
Use the flag --livereload|-l for LiveReload.

gulp test
=========
Runs the unit tests.
A html test report file is generated in tests/test-report.
A html coverage report file is generated in tests/coverage/PhantomJS 2.1.1 (Mac OS X 0.0.0)
