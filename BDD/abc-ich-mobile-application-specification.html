For the purposes of the calculatuion the weight is rounded to the nearest 10 Kg. If the weight is above 100Kg then it is fixed at 100Kg. If the weight is below 30Kg then it is fixed at 30Kg. The calculated value depends on which range the INR value falls within as follows: (1.3 - 3.9): dose = weightInKg x 25 (4.0 - 6.0): dose = weightInKg x 35 (> 6.0): dose = weightInKg x 50 #1: Calculation of Beriplex dose with Inr value Given: the user has entered valid data on the "Calculate Beriplex" view And: the user has selected "No" to the question "Do you plan to administer Beriplex without waiting for INR?"
